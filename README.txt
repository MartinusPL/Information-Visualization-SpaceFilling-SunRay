This is an academic project created for purposes of Visialization of Information subject at Gdañsk University of Technology, ETI Faculty.

The aim of this project is to generate chart named Sun Ray. It's a type of Space Filling charts that represents chierarchical values.

In the sample_input.csv file you are given sample input data for the application. It contains data about car sales (generated randomly) in General Motors. It represents chierarchical structure of Concern->Type->Mark->Model->Quantity (that is prezented on the graph generated by the application).
